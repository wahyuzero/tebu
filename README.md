# TEBU — Fruit Word Guessing Game 🍎

A simple Indonesian word game where players arrange scrambled letters to guess fruit names.

## Features

- 8 fruit levels with increasing difficulty
- Learn mode & Play mode
- Offline — all assets bundled, no server needed
- Works on phone, tablet, and web
- Kid-friendly UI with sound effects & background music

## Play Online

[tebu.frugaldev.biz.id](https://tebu.frugaldev.biz.id)

## Build APK

```bash
npm install
npm run build
npx cap sync android
cd android && ./gradlew assembleDebug
```
